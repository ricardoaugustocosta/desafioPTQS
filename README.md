### Desafio 5 do PTQS

### Objetivo:
Projeto para o desafio de desenvolver um teste de automação em um E-commerce.

### Pré requisitos
Para executar este projeto, você precisará de:

	• IntelliJ IDEA Community Edition 2021.3.2
	• Module JDK: 1.8
	• JUnit (5.8.2)
	• WebDriver (4.1.2)
	• Google Chrome (100.0.4896.88)
	• ChromeDriver (100)
	• Git (2.33.0)
